
# 🚀 Smart Unit Converter

A modern and responsive unit conversion web application built with Python (Flask). This tool provides instant conversions for various categories like Length, Mass, and Temperature with a clean, user-friendly interface.

## ✨ Key Features
- Dynamic UI: Units update automatically based on the selected category without page refresh.
- Full-Stack Logic: Backend processing with Flask and frontend interactivity with JavaScript.
- Responsive Design: Works perfectly on both desktop and mobile devices.
- Error Handling: Robust logic to handle different unit types and conversion formulas.

## 🛠️ Technology Stack
- Backend: Python 3.x, Flask
- Frontend: HTML5, CSS3, JavaScript (ES6)
- Styling: Custom CSS with a modern dark/clean theme.

## 🚀 How to Run Locally
1. Clone this repository:
   git clone https://github.com/mahdiyar1991/smart-unit-converter.git
   
2. Install dependencies:
   pip install flask
   
3. Run the app:
   python main.py
   
4. Visit `http://127.0.0.1:5000` in your browser.
